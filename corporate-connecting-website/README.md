# Corporate Connecting - Landing Page

This is a React + Vite single-page application for Corporate Connecting, an office furniture store in Bengaluru.

## Setup Instructions

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

## Customization

Search the codebase for `<!-- REPLACE: ... -->` comments to update the placeholders with actual business information:
- Phone numbers
- Email addresses
- Working hours
- Google Maps embed URL

## Tech Stack
- React 18
- Vite
- Tailwind CSS v4
- Lucide React (Icons)
- Motion (Framer Motion)
